# Добавление шаблонов процедур в MCreator

Шаблоны процедур (.ptpl) позволяют быстро добавлять готовые логические блоки в редактор процедур.

## Структура шаблона

Файл шаблона — это стандартный XML-файл Blockly, сохраненный с расширением `.ptpl`.

### Расположение файлов
- **Системные шаблоны**: `plugins/<plugin_name>/templates/procedures/`
- **Пользовательские шаблоны**: `<user_home>/.mcreator/templates/procedures/`

## Инструкция по добавлению

1. **Создайте логику**:
   - Откройте редактор процедур в MCreator.
   - Соберите нужную комбинацию блоков.
2. **Экспортируйте шаблон**:
   - Нажмите кнопку **Export** в верхней панели редактора Blockly.
   - Сохраните файл с расширением `.ptpl`.
3. **Добавьте локализацию**:
   - Для перевода названия шаблона добавьте ключ в файлы `.properties` плагина или в настройки локализации.
   - Формат ключа: `template.procedure.<filename_without_extension>=Название на русском`
   - Пример для `light_toggle.ptpl`:
     `template.procedure.light_toggle=Переключить свет блока`

## Требования к Fabric
При создании шаблонов для Fabric убедитесь, что вы используете блоки, поддерживаемые этим генератором (например, блоки работы со свойствами состояний блоков для переключения света).
