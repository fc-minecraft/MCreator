# Процедуры

Процедуры позволяют добавлять логику в ваш мод без написания кода на Java.

MCreator использует систему визуального программирования (похоже на Scratch). Вы соединяете блоки логики, чтобы определить поведение.

Примеры использования:
* Выдать эффект зелья при съедении предмета.
* Взрыв при разрушении блока.
* Спавн моба при нажатии клавиши.
