# Интерфейс

Главное окно MCreator состоит из нескольких частей:

* **Меню**: Файл, Правка, Ресурсы и т.д.
* **Панель инструментов**: Быстрый доступ к созданию элементов, запуску клиента и экспорту.
* **Список элементов**: Все созданные вами блоки, предметы и т.д.
* **Консоль**: Вывод Gradle и логов игры.
