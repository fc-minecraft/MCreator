# Начало работы

Чтобы начать создавать моды, вам нужно создать рабочее пространство.

1. Запустите MCreator.
2. Нажмите "Новое рабочее пространство".
3. Введите имя мода и выберите тип генератора (например, для мода Forge).
4. Нажмите "Создать новое рабочее пространство".

MCreator настроит окружение. Это может занять некоторое время.
