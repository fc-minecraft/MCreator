Нужно ли рисовать красивые частицы вокруг блока?

Этот триггер срабатывает постоянно, пока игрок смотрит на блок. Используй его, чтобы:
*   Создавать частицы дыма или магии.
*   Проигрывать тихие звуки.
*   **Важно:** Не меняй здесь мир (не ломай блоки, не спауни мобов), это работает только для красоты (на клиенте)!
