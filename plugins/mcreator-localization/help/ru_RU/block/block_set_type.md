Из чего сделан твой блок? Это влияет на звуки и инструменты.

*   **Stone** (Камень) — нужен кирка, звук камня.
*   **Wood** (Дерево) — нужен топор, звук дерева.
*   **Iron** (Железо) — звонкий звук металла.
*   **Earth** (Земля) — глухой звук, копается лопатой.

Выбирай то, что больше всего подходит по смыслу!
