Как мобы будут воспринимать твой блок?

*   **Проходимый** (Walkable) — по нему можно ходить (как земля).
*   **Забор** (Fence) — это препятствие, через него не перепрыгнуть.
*   **Люк** (Trapdoor) — это люк, он может открываться.
*   **Заблокирован** (Blocked) — сюда ходить нельзя (как стена).

Это помогает мобам (зомби, жителям) понимать, как обходить твой блок.
