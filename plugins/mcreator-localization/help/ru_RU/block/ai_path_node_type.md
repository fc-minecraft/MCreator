Как мобы будут воспринимать твой блок?

*   **WALKABLE** — по нему можно ходить (как земля).
*   **FENCE** — это забор, через него не перепрыгнуть.
*   **TRAPDOOR** — это люк, он может открываться.
*   **BLOCKED** — сюда ходить нельзя (как стена).

Это помогает мобам (зомби, жителям) понимать, как обходить твой блок.
