Выберите таблицы добычи для случайного распределения между сундуками в этой структуре.

Когда в рамках этой структуры генерируется `minecraft:chest`, генератор автоматически подменяет его NBT-данные на одну из выбранных таблиц лута.
Если выбрано несколько таблиц, они будут распределяться с равной вероятностью.
