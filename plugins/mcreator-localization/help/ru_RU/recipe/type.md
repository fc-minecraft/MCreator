Где будет работать твой рецепт? Выбери нужную печку или стол!

*   **Crafting** (Крафт) — для обычного Верстака (3x3).
*   **Smelting** (Плавку) — для обычной Печки (руда -> слиток, еда -> жареная еда).
*   **Blasting** (Плавильная печь) — только для руд и металлов (быстрее обычной печи).
*   **Smoking** (Коптильня) — только для еды (быстрее обычной печи).
*   **Stone cutting** (Камнерез) — для резки камня (1 камень -> 1 ступенька).
*   **Campfire cooking** (Костёр) — для жарки еды на костре (без угля!).
*   **Smithing** (Стол кузнеца) — для улучшения брони (например, алмазная -> незеритовая).
*   **Brewing** (Варка) — для зельеварки.
