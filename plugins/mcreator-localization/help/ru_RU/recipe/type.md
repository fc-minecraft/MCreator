Где будет работать твой рецепт? Выбери нужную печку или стол!

*   **Крафт** (Crafting) — для обычного Верстака (3x3).
*   **Плавку** (Smelting) — для обычной Печки.
*   **Плавильная печь** (Blasting) — только для руд (быстрее).
*   **Коптильня** (Smoking) — только для еды (быстрее).
*   **Камнерез** (Stone cutting) — для резки камня.
*   **Костёр** (Campfire cooking) — для жарки еды на костре.
*   **Стол кузнеца** (Smithing) — для улучшения брони.
*   **Варка** (Brewing) — для зельеварки.
