В какой вкладке зелёной книги рецептов игроки найдут твой предмет?

*   **Building** (Строительство) — для блоков (как доски).
*   **Redstone** (Редстоун) — для механизмов (как поршни).
*   **Equipment** (Снаряжение) — для мечей, брони и инструментов.
*   **Misc** (Разное) — для всего остального (как красители).

Выбери правильную категорию, чтобы рецепт не потерялся!
