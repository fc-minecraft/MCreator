Это уникальный ID (идентификатор) твоего рецепта. Игра использует его, чтобы не путаться.

*   **Как назвать:** Используй только маленькие английские буквы и `_` (например, `super_sword`).
*   **Правило:** В твоём моде не должно быть двух рецептов с одинаковым именем в реестре!

Если ты хочешь переопределить рецепт Майнкрафта (например, рецепт факела), напиши здесь **torch**, а в поле "Домен рецепта" выбери **minecraft**.
