Редактор интерфейса (GUI)

Здесь создаются меню для сундуков, печек, машин и просто информационные окна.

Компоненты

  • Slot (Слот): Ячейка для предмета.
    • Input: Сюда кладут.
    • Output: Отсюда забирают (нельзя положить).
  • Button (Кнопка): Нажми меня! Можно привязать Процедуру к нажатию.
  • Text (Текст): Просто надпись.
  • Image (Картинка): Декор.

Привязка к блоку

Чтобы GUI открывался при клике на блок, нужно в настройках Блока (вкладка "Tile Entity") поставить галочку "Has GUI" и выбрать этот интерфейс.
